Why EOS.io Software?

The EOS.IO software introduces a new blockchain architecture designed to enable vertical and horizontal scaling of decentralized applications. This is achieved by creating an operating system-like construct upon which applications can be built. The software provides accounts, authentication, databases, asynchronous communication and the scheduling of applications across many CPU cores or clusters. The resulting technology is a blockchain architecture that may ultimately scale to millions of transactions per second, eliminates user fees, and allows for quick and easy deployment and maintenance of decentralized applications, in the context of a governed blockchain.

Source : https://developers.eos.io/eosio-home/docs/what-is-eos

Polaris

What is the Polaris?
Polaris is a Public Blockchain Network based on EOS.io that adds several features such as Multi-Party Computation(MPC) and Zero-Knowledge Proof, enhances accessibility for the public, and provides a platform for enterprises to participate in the Polaris Network.
Who maintains "Polaris.io Software"?
Polaris.io Software is based on EOS.io Software, and the overall technology development is under the heading of "Outsourcing company name".
When will the Polaris Network be launched?
We plan to activate the test network within 2018, and the main network by 2019 with the participation of Enterprise Block Producers.

Technical

How does "Multi-Party Computation" work?
As the name suggests, it distributes computation tasks so that the verification of each task becomes easier.
How does the Zero-Knowledge Proof apply to Polaris?
Zero-Knowledge Proof acts as a processing layer in the form of Smart Contract. This enables transaction anonymization without the need of changing the client code.

Token

When is Airdrop and Snapshot?
Snapshot is performed based on the Genesis Block of EOS. The actual distribution takes place when the Polaris Network is activated.
What wallet should I use to store the Polar tokens?
Polaris Foundation will provide Polaris Wallet Software upon the activation of Polaris Network.
When will the ERC-20 based Token be activated on the Polaris Network?
Prior to a month of the official launch of the Polaris Network, ERC-20 based Polar Tokens will be fixed (non-transferable) on the Ethereum blockchain. ERC-20 based Polar token holders will receive native Polar Tokens upon the activation of the Polaris Network.
On which blockchain network will Polar token be transmitted?
Before the Polaris Network is activated, you can transfer the ERC-20 type Polar Token on the Ethereum main network. Once the Polaris Network is activated, the ERC-20 type Polar Token will be fixed (non-transferable).
